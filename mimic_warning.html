Attention: Action Required

Your Flatpak version does not support the expose-pids flag, which means that this Brave Flatpak will have to run in a less performant / efficient method. (This generally happens if your distro's bubblewrap installation is setuid instead of using user namespaces.)

Not having support for the expose-pids flags will cause printing, PDF viewing, and extensions to stop functioning properly.

Therefore it is important for you to enable user namespaces.

In order to resolve this, run the following commands as root: 

echo 'kernel.unprivileged_userns_clone=1' >> /etc/sysctl.conf
sysctl -p
bash -c 'chmod -s $(which bwrap)'

Afterwards, restart your computer to make sure your flatpak is running properly.
